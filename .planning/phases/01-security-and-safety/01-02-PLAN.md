Replace the body of `free_http_request()` with:
```c
void free_http_request(HTTPRequest *req) {
    if (!req) return;

    // Free request line fields (now strndup'd copies)
    free(req->request_line.method);
    free(req->request_line.uri);
    free(req->request_line.protocol);

    // Free each header's name and value (now strndup'd copies)
    if (req->headers) {
        for (int i = 0; i < req->header_count; i++) {
            free(req->headers[i].name);
            free(req->headers[i].value);
        }
        free(req->headers);
    }

    // Free body if present
    free(req->body);

    free(req);
}
```

**IMPORTANT:** `free(NULL)` is safe in C (no-op), so NULL checks on individual fields before free are optional but harmless. The above pattern is correct.
